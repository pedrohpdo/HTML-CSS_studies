Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir para dar semântica
aos elementos do seu website. Passe o mouse sobre os termos para descobrir quais
tags devem ser usadas.

  • Paragrafos
  • Títulos
  • Imagens
  • Abreviações
  • Código Fonte
  • Links
  • Listas Numeradas
  • Listas não numeradas
  • Listas com Demarcadores
