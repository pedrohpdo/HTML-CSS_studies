ps: Esse link acima, do jeito que está configurado, pode abri um precedente para
que seu usuário jamais volte a sua página inicial, já que, ao acessar o link a página
atual é sobreposta. Para solucionar isso, podemos adicionar duas propriedades a tag
de links. As propriedas target = blank e rel = external.

Traduzindo:

Target Blank
Propriedade que indica que, o alvo desse meu link será uma nova janela em branco.
Rel External
Estou inferindo que o link em questão vai para um lugar fora do meu site

Links para Download

Agora vamo humildemente entender links para downloads. Arrocha.

Criando link para download de PDF's

Existem particularidades para determinados navegadores, mas geralmente podemos
inferir o download de arquivos em links com a mesma tag de links, mas referenciando
o arquivo em questão (o mesmo deve estar dentro da sua pasta). Caso não funciona,
também é possível adicionar duas novas propriedades dentro da tag, a download = nomeDoArquivo
e o type = tipoDeMediaDoArquivo.

Com relação a propriedade type, existem uma séria de media types para determinado tipo
de arquivo e vc pode acessa-las aqui
