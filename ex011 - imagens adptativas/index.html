Trabalhando com Imagens Adptativas

Para isso, vamos utilizar a tag <picture>, onde ela infere que uma imagem será
inserida, mas com algumas propriedes, como a souce-media-type

ps: Usando o source, setamos na propriedade media o tamanho máximo de tela em questão. Ou seja,
caso o tamanho da tela atual seja de, no máximo 1050px (Ou menor que 1050 px), a imagem Prancheta2
deve ser mostrada.
Caso o tamanho da tela atual seja de, no máximo 750px (ou menor que 750), utilizamos a imagem Prancheta1
deve ser mostrada no lugar.
Existem 50px excedentes no max-width que foram colocando para evitar que a barra de rolagem lateral esteja
habilitada, mesmo sendo desnecessário pelo tamanho da imagem.

Cuidado: Invertendo os Sources

Os sources devem estar em ordem de tamanho, caso contrário o script não vai conseguir mostrar as fotos com
a devida responsividade.
