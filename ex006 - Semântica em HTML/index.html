O que é Semântica

Semântica é significado, tem a ver com sematologia
O significado das palavras, que tem mais valor que sua forma.

Comparando com HTML 4, algumas tags passaram a ficar em desuso,
uma vez que tags antigas não tinha significado, mas sim forma. Para
isso, foi transferido tudo isso para o CSS

HTML diz sobre o Significado do Conteúdo

Cores, formas, estilos e coisas desse tipo não são mais cabíveis dentro
do HTML e NÃO É NEM UM POUCO RECOMENDÁVEL você usar isso.
Não seja um pateta.
