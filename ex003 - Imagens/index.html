Inserção de Imagens Usando HTML

Lembrar sempre de DIREITOS AUTORAIS DE IMAGEM

Agora, vamo entender como colocar imagens utilizando Tags HTML
Caso a imagem esteja dentro do src do index.html, é possível inseri-la apenas referenciando seu nome na tag img

Dá pra carregar imagens que estão em sub pastas
Dentro de sub pastas, basta simplesmente mostrar o path até a imagem requisitada

E agora, também é possível utilizar imagens externas, se liga negão
É só pegar a URL da imagem na web e um abraço
